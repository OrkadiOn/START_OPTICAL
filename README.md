# START_OPTICAL

Автоматическая настройка оптического терминала Eltex NTU-RG-1402G-W, если после отключения электричества терминал сбрасывается в заводские настройки.

Настройка через http, так как telnet на терминале в default выключен, и имеет кривую настройку pppoe

В реальной схеме использовалась настройка оптического терминала, с подключенного после него в локальной сети роутера на OpenWRT.
